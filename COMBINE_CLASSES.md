# Combine Classes Feature

## Overview
Added a "Combine Classes" feature to allow viewing students from multiple classes together - perfect for when two or more classes join for a lesson.

## How It Works

### 1. Starting Combine Mode
- On the class selection screen, tap the **üîó Combine Classes** button
- The screen switches to selection mode with checkboxes on each class

### 2. Selecting Classes
- Tap any class button to toggle its checkbox
- OR tap the **‚úì Select All** button to select all classes at once
  - Tapping "Select All" again will deselect all
- A live count shows total students as you select

### 3. Viewing Combined Roster
- Once you've selected your classes, tap **View Combined (X total students)**
- The student roster will show all students from the selected classes
- The header will display:
  - If all classes selected: **"Everyone"**
  - Otherwise: **"Class A + Class B + Class C"**

### 4. Canceling
- Tap **‚Üê Cancel** to exit combine mode and return to normal class selection
